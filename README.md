# NEOSA
A web application to help students find fellow colleagues to study and learn with.

#Website
http://www.neosa-uiuc.appspot.com

#To develop:
* Clone the repository.
* Add the existing project using the GoogleAppEngineLauncher.
* `cd neosa-uiuc` from the base repository directory.
* `bower install` to install dependencies.
* `bower install --save <dependency>` to add and install new dependencies.
